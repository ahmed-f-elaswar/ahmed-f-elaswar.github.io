Design of Web-Based System: Assignment 1Personal CV Webpage
Introduction
In this assignment, you will design and develop a personal CV (Curriculum Vitae) webpageusing HTML and CSS, and publish it online.
The goal of this assignment is to help you learn the fundamentals of webpage structure, styling,and online publishing by creating your own professional portfolio page.
Your webpage should present your personal and academic profile in a clean and professionaldesign. You are encouraged to customize the layout, colors, and fonts to reflect your own stylewhile maintaining good usability and readability.
You are also required to publish your webpage online using a free hosting service such asGitHub Pages, Netlify, or any similar platform, and submit the public link.
Deadline: 16 March 2026.
Your submission should meet the following requirements:
1. Webpage Content
Create an HTML webpage that contains at least the following CV sections:
 Full Name and Professional Title
 Profile / Short Bio
 Education
 Skills
 Projects or Experience
 Certifications or Achievements (if any)
 Contact Information (email, LinkedIn, etc.)
2. Page Design
 Use a separate CSS file to style the webpage.
 Organize the content using appropriate HTML tags and semantic structure.
 Customize the colors, fonts, and layout to create a professional design.
 Ensure the page has clear spacing and readable formatting.
3. Images
Include at least one personal image such as:
 A profile picture
 Or a professional avatar
 
 
 
4. Online Publishing
Students must publish their webpage online using any free hosting service, such as:
 GitHub Pages
 Netlify
 Vercel
 Any other static website hosting service
The webpage must be publicly accessible via a URL.
Submission Instructions
Submit your assignment as a ZIP file containing the following:
 HTML file (e.g., index.html) containing the CV structure.
 CSS file (e.g., styles.css) used for styling the page.
 Any assets used (images, icons, etc.).
 A text file named link.txt that contains the public URL of your published webpage.
Example structure:
StudentName_Assignment1.zip│├── index.html├── styles.css├── images/│ └── profile.jpg└── link.txt
